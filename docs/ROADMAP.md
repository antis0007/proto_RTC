# Roadmap

## Phase 1: Text baseline (current skeleton)

- Login, guild/channel listing, WS relay
- Ciphertext blob schema and relay semantics
- Basic moderation checks

## Phase 2: Encrypted files

- Ciphertext file upload/download
- Metadata indexing and retention policy

## Phase 3: Voice control-plane integration

- Stable token minting, room policies
- Better room lifecycle UX

## Phase 4: Screen share

- Channel capabilities
- Fine-grained publish grants

## Phase 5: P2P/E2EE hardening

- Real key management interfaces
- Device trust + session recovery
- Optional P2P optimizations where feasible
